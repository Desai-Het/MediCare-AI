# MediCare-AI
MediCare-AI is an AI-powered medical assistant that provides healthcare guidance by answering user questions using large language models and contextual medical knowledge, with a focus on safety, clarity while encouraging professional consultation when needed.

# How to run?
### STEPS:

Clone the repository

```bash
git clone https://github.com/Desai-Het/MediCare-AI.git
```

### STEP 01 - Create a conda environment after opening the repository

```bash
conda create -n medibot python=3.10 -y
```

```bash
conda activate medibot
```

### STEP 02 - Install the requirements
```bash
pip install -r requirements.txt
```
